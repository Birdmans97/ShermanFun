# ShermanFun

A Next.js website built with TypeScript.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

The website will be available at [http://localhost:80](http://localhost:80).

## Available Scripts

- `npm run dev` - Start the development server on port 80
- `npm run build` - Build the production version
- `npm start` - Start the production server on port 80
- `npm run lint` - Run ESLint

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS3
